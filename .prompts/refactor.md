# 代码重构

> **分类**：`.prompts/`
> **适用场景**：对功能正确但代码质量较差的代码进行重构，提升可读性和可维护性，同时保持行为不变。
> **版本**：1.0

---

## 描述

提供待重构的代码和重构目标，AI 将给出重构后的代码并解释每处改动的原因。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入 `{{编程语言}}`、`{{代码片段}}` 和 `{{重构目标}}`。

## Prompt 模板

```
请对以下 {{编程语言}} 代码进行重构，保持功能不变的前提下提升代码质量。

## 原始代码

```{{编程语言}}
{{代码片段}}
```

## 重构目标

{{重构目标}}

## 重构要求

1. **保持行为一致**：重构前后功能必须完全相同
2. **逐步重构**：如果改动较大，请分步骤说明重构思路
3. **解释改动**：对每一处关键改动说明原因
4. **遵循原则**：优先应用 SOLID、DRY、KISS 原则

## 输出格式

1. 重构思路概述
2. 重构后的完整代码
3. 关键改动对照表（原代码 → 新代码 + 原因）
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{编程语言}}` | 目标语言 | `Python`、`Java` |
| `{{代码片段}}` | 待重构的代码 | *(粘贴代码)* |
| `{{重构目标}}` | 重构的具体目标 | `消除嵌套 if、提取重复逻辑为工具函数、改善命名` |
