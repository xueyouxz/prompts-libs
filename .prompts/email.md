# 专业邮件撰写

> **分类**：`.prompts/`
> **适用场景**：快速撰写各类专业场景邮件，包括商务沟通、跟进催促、申请请求等。
> **版本**：1.0

---

## 描述

提供邮件的场景、目的和关键信息，AI 将生成语气得体、逻辑清晰的专业邮件。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入各变量，特别是 `{{邮件目的}}` 和 `{{关键信息}}`。

## Prompt 模板

```
请帮我起草一封专业邮件。

## 基本信息

- **发件人身份**：{{发件人身份}}
- **收件人身份**：{{收件人身份}}
- **邮件目的**：{{邮件目的}}
- **语气要求**：{{语气要求}}（如：正式 / 友好专业 / 简洁直接）
- **语言**：{{语言}}

## 邮件内容要点

{{关键信息}}

## 写作要求

1. 主题行简洁有力，能清楚传达邮件目的
2. 开头简短寒暄（如适合）
3. 正文逻辑清晰，段落分明
4. 结尾包含明确的下一步行动或期望回复
5. 控制在 {{字数}} 字以内

## 输出格式

主题：[邮件主题]

[邮件正文]
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{发件人身份}}` | 我的角色或职位 | `产品经理` |
| `{{收件人身份}}` | 对方的角色或职位 | `外部供应商` |
| `{{邮件目的}}` | 发送这封邮件要达到的目标 | `催促对方提交报价方案` |
| `{{语气要求}}` | 邮件的语气风格 | `友好且专业` |
| `{{语言}}` | 邮件语言 | `中文` 或 `English` |
| `{{关键信息}}` | 需要在邮件中包含的关键内容 | `截止日期是本周五，项目预算约 50 万` |
| `{{字数}}` | 字数上限（可选） | `300` |
