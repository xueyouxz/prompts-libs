# 头脑风暴

> **分类**：`.prompts/`
> **适用场景**：围绕一个主题快速发散思维，生成大量创意想法，打破思维定势。
> **版本**：1.0

---

## 描述

提供一个问题或主题，AI 将模拟多角色头脑风暴，生成多样化、有创意的想法列表，并进行分类和优先级排序。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入 `{{头脑风暴主题}}` 和约束条件。
3. 可以多轮迭代，要求 AI 对某个方向深度展开。

## Prompt 模板

```
请围绕以下主题进行头脑风暴，帮我生成尽可能多样化的创意想法。

## 主题

{{头脑风暴主题}}

## 背景与约束

- **目标**：{{目标描述}}
- **约束条件**：{{约束条件}}（如预算、时间、技术限制等）
- **想法数量**：至少 {{数量}} 个

## 头脑风暴规则

1. **发散优先**：先不评判好坏，尽量多生成想法
2. **多角度思考**：从不同用户视角、不同技术路径、不同商业模式等维度切入
3. **鼓励大胆**：包含 2-3 个"疯狂"但有潜力的想法
4. **建立连接**：尝试组合不同领域的概念

## 输出格式

1. **想法清单**（按类别分组）
2. **TOP 3 推荐**（附理由）
3. **可快速验证的小实验**（如何用最小成本验证某个想法）
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{头脑风暴主题}}` | 要展开思考的主题或问题 | `如何提高开发团队的工程效率` |
| `{{目标描述}}` | 希望通过这些想法达成什么 | `找到 3 个可在下季度落地的效率优化方案` |
| `{{约束条件}}` | 需要遵守的限制 | `不增加人力，不依赖采购新工具` |
| `{{数量}}` | 期望生成的想法数量 | `20` |
