# 数据分析报告

> **分类**：`prompts/analysis/`
> **适用场景**：对数据集或分析结果进行解读，生成结构化的数据分析报告。
> **版本**：1.0

---

## 描述

提供原始数据或数据摘要，AI 将帮助进行模式识别、趋势分析和结论提炼，输出可直接使用的分析报告。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入数据和分析目标。
3. 如果数据量大，先提供数据摘要（如统计量）而不是原始数据。

## Prompt 模板

```
请对以下数据进行分析并生成报告。

## 分析背景

- **业务场景**：{{业务场景}}
- **分析目标**：{{分析目标}}
- **受众**：{{报告受众}}（如：技术团队 / 管理层）

## 数据

```
{{数据内容}}
```

## 分析要求

1. **描述性分析**：关键指标的基本统计（均值、趋势、分布）
2. **模式识别**：找出数据中的规律、异常点或相关性
3. **结论提炼**：基于数据得出 3-5 条关键结论
4. **行动建议**：针对结论给出可操作的业务建议
5. 用图表描述替代实际图表（如"可制作折线图展示……"）

## 输出格式

请按以下结构输出：
1. 执行摘要（3 句话）
2. 数据概览
3. 关键发现
4. 结论与建议
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{业务场景}}` | 数据的业务背景 | `电商平台 2024 年 Q4 销售数据` |
| `{{分析目标}}` | 期望从分析中得到什么 | `找出销售额下降的原因` |
| `{{报告受众}}` | 报告的阅读对象 | `产品负责人` |
| `{{数据内容}}` | 原始数据或数据摘要 | *(粘贴数据或统计摘要)* |
