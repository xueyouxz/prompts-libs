# 代码调试助手

> **分类**：`prompts/coding/`
> **适用场景**：遇到 bug 或异常错误时，快速定位问题根因并获得修复方案。
> **版本**：1.0

---

## 描述

提供错误信息和相关代码，让 AI 帮助分析 bug 成因、给出修复方案，并解释如何避免同类问题。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入 `{{错误信息}}`、`{{代码片段}}` 和 `{{已尝试方案}}`。
3. 可选：补充 `{{运行环境}}` 以缩小排查范围。

## Prompt 模板

```
我在调试以下代码时遇到了问题，请帮我分析原因并给出修复方案。

## 错误信息

```
{{错误信息}}
```

## 相关代码

```{{编程语言}}
{{代码片段}}
```

## 运行环境

{{运行环境}}

## 已尝试的方案

{{已尝试方案}}

## 期望结果

{{期望结果}}

---

请按以下步骤回答：
1. **根因分析**：解释错误产生的根本原因
2. **修复方案**：给出完整的修复后代码
3. **验证方法**：如何验证修复是否有效
4. **预防建议**：如何避免此类问题再次出现
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{错误信息}}` | 完整的错误堆栈或错误描述 | `TypeError: Cannot read properties of undefined` |
| `{{编程语言}}` | 代码所用语言 | `JavaScript` |
| `{{代码片段}}` | 出现问题的代码 | *(粘贴代码)* |
| `{{运行环境}}` | OS、语言版本、框架版本等 | `Node.js 20, macOS 14` |
| `{{已尝试方案}}` | 已经尝试过的修复思路 | `加了 null 检查，问题依然存在` |
| `{{期望结果}}` | 正确运行时应有的输出 | `返回用户对象` |
