# 博客文章创作

> **分类**：`prompts/writing/`
> **适用场景**：根据主题和受众快速生成结构完整、逻辑清晰的博客文章草稿。
> **版本**：1.0

---

## 描述

提供文章主题、目标读者和风格要求，AI 将生成包含大纲和正文的完整博客文章草稿。

## 使用方法

1. 复制下方模板到 AI 对话框。
2. 填入各变量。
3. 先让 AI 确认大纲，满意后再生成正文，避免大段返工。

## Prompt 模板

```
请帮我撰写一篇博客文章。

## 文章信息

- **主题**：{{文章主题}}
- **目标读者**：{{目标读者}}
- **文章风格**：{{文章风格}}（如：技术深度文、入门科普、观点评论）
- **字数要求**：约 {{字数}} 字
- **核心观点/亮点**：{{核心观点}}

## 写作要求

1. 标题要吸引人，可给出 3 个备选标题
2. 开篇用问题或数据引发读者兴趣
3. 结构清晰，使用小标题分隔段落
4. 结尾有明确的行动号召（Call to Action）
5. 语言风格：{{语言风格}}（如：专业严谨 / 轻松幽默 / 简洁直白）

## 输出格式

1. 3 个备选标题
2. 文章大纲
3. 完整正文
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{文章主题}}` | 文章的核心主题 | `使用 GitHub Copilot 提升开发效率` |
| `{{目标读者}}` | 读者画像 | `有 1-3 年经验的前端开发者` |
| `{{文章风格}}` | 文章类型 | `技术实践分享` |
| `{{字数}}` | 目标字数 | `1500` |
| `{{核心观点}}` | 文章的核心论点或亮点 | `AI 编程工具改变了开发者的工作流` |
| `{{语言风格}}` | 写作语气 | `专业且易读` |
